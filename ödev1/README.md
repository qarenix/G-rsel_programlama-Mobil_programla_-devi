# Varlik Yonetimi Odev Projesi

Bu proje Java dili kullanilarak Android Studio'da hazirlanmistir. Amac, bir isletmede veya okulda bulunan demirbas/varlik kayitlarini basit sekilde takip edebilen bir mobil uygulama yapmaktir.

## Projede Yapilanlar

- Varlik ekleme
- Varlik duzenleme
- Varlik silme
- Konut, arsa, elektronik, esya, arac, nakit gibi tur secme
- Secilen ture gore gerekli alanlari gosterme
- Kategoriye gore filtreleme
- Arama yapma
- Toplam kayit ve toplam deger bilgisi gosterme
- Aylik gelir ve aylik gider takibi
- Harcama, alisveris ve yatirim kaydi ekleme
- Para alanlarini 1.000.000 seklinde nokta ile gosterme
- Kayitlari telefon hafizasinda saklama

## Kullanilan Teknolojiler

- Android Studio
- Java
- SharedPreferences
- JSON
- Gradle

## Calistirma

1. Android Studio acilir.
2. Bu proje klasoru `File > Open` ile secilir.
3. Gradle Sync islemi bitince uygulama `Run` tusu ile calistirilir.

APK dosyasi build alindiktan sonra su klasorde olusur:

`app/build/outputs/apk/debug/app-debug.apk`
